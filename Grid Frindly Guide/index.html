Grid Container Basics

display: grid

Creates a grid container. All direct children become grid items.

grid-template-columns

Defines the columns of the grid.

grid-template-rows

Defines the rows of the grid.

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
}

Item Placement

grid-column & grid-row

Controls where items are placed using line numbers.

grid-area

Shorthand for grid-row-start, grid-column-start, grid-row-end, and grid-column-end.

.item-1 {
  grid-column: 1 / 3; /* Start at line 1, end at line 3 */
  grid-row: 1 / 2;
}

.item-2 {
  grid-area: 1 / 3 / 3 / 4; /* row-start / col-start / row-end / col-end */
}

/* Responsive photo gallery with CSS Grid */
.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));
  grid-auto-rows: 150px;
  gap: 10px;
}

.portrait {
  grid-row: span 2;
}

.wide {
  grid-column: span 2;
}
